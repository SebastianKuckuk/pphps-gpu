# PPHPS - GPU Programming with OpenMP Target Offloading

This repository collects the materials for the *GPU Programming with OpenMP Target Offloading* part of the [NHR@FAU](https://nhr.fau.de) course on *Parallel Programming of High-Performance Systems (PPHPS)*.
Start by cloning it
```bash
git clone https://github.com/SebastianKuckuk/pphps-gpu
```

Afterwards navigate to the [target-offload](target-offload.ipynb) notebook for further guidelines.
